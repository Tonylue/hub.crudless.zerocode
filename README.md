## CrudlessHub
`云应用**无CRUD服务**定义编排中心`

### 通过编排文件 crudless.yml  定义应用 
直接基于业务理解定义应用服务，各实体单元包括列表以详情字段，还可以自动构建数据库sql, 自动生成API代码

[编排定义源文件(crudless.yml)](crudless.yml)
